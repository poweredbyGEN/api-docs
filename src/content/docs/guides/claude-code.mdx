[Claude Code](https://docs.anthropic.com/en/docs/claude-code) is Anthropic's CLI for Claude. It supports **MCP (Model Context Protocol)** servers, which let Claude interact with external APIs through natural language. The Auto Content Engine MCP server gives Claude direct access to your GEN agents, content engines, rows, cells, generations, organizations, and assets.

<Aside type="tip">
  Source code: [github.com/poweredbyGEN/autocontentengine-mcp-server](https://github.com/poweredbyGEN/autocontentengine-mcp-server)
</Aside>

## Setup

<Steps>

1. **Get your API key**

   Sign in to [gen.pro](https://gen.pro), go to **Settings → API Keys**, and click **Create API Key**. Copy the key immediately — it's only shown once.

   ```
   ref_a1b2c3d4e5f6...
   ```

   <Aside type="caution">
     If you lose your key, revoke it and create a new one. Keys cannot be retrieved after creation.
   </Aside>

2. **Install the MCP server**

   ```bash
   npm install -g @poweredbygen/autocontentengine-mcp-server
   ```

3. **Add to Claude Code**

   Add the server to `~/.claude/mcp.json` (global) or `.mcp.json` (per-project):

   ```json
   {
     "mcpServers": {
       "gen-autocontentengine": {
         "command": "autocontentengine-mcp-server",
         "env": {
           "GEN_API_KEY": "ref_a1b2c3d4e5f6..."
         }
       }
     }
   }
   ```

4. **Restart Claude Code**

   The MCP server loads on startup. After restarting, Claude will have access to 38+ `gen_*` tools.

</Steps>

## What you can do

Once configured, manage your entire content pipeline through natural language:

| Ask Claude... | What happens |
|--------------|--------------|
| "Show me my agents" | Lists all your GEN agents with their workspaces |
| "Create a new content engine called March Campaign" | Creates an engine under your agent |
| "Add 5 rows with scripts about morning routines" | Creates rows and populates script cells |
| "Generate a Veo 3 video for row 1" | Triggers AI video generation |
| "What's the status of my generation?" | Polls until complete, shows output URL |
| "List my organizations and credit balance" | Shows workspaces with available credits |
| "Upload this image to my asset library" | Handles the direct upload workflow |

## Available tools

The MCP server provides tools across these categories:

| Category | Tools | Examples |
|----------|-------|---------|
| **Discovery** | 3 | List agents, workspaces, get user profile |
| **Agents** | 7 | Create, update, delete agents and avatars |
| **Organizations** | 5 | Manage workspaces, check credits |
| **Content Engines** | 3 | Create, get, clone engines |
| **Rows & Columns** | 5 | Create rows, list/create columns |
| **Cells & Layers** | 4 | Get/update cells, create/delete layers |
| **Generations** | 4 | Generate content, check status, stop jobs |
| **Content Resources** | 7 | Upload files, browse asset library |

Claude also has access to a built-in **API reference resource** — it can read the full API documentation on demand to understand generation types, parameters, and workflows.

## Generation types

When Claude triggers content generation, it uses these `generation_type` values:

| Content | generation_type | Models |
|---------|----------------|--------|
| **Text** | `text_generation` | Gemini, OpenAI |
| **Image** | `gemini_image_generation` | Gemini, Gemini Pro |
| **Image** | `midjourney` | Midjourney |
| **Video** | `gemini_video_generation` | Veo 3, Veo 3.1 |
| **Video** | `sora2_video_generation` | Sora 2 |
| **Video** | `kling` | Kling 1.6 |
| **Video** | `seedance_video_generation` | Seedance Pro |
| **Speech** | `eleven_labs` | ElevenLabs |
| **Lipsync** | `lipsync` | sync.so, gen |
| **Captions** | `captions` | Gemini |

See the [Creation Cards](/reference/creation-cards/) reference for full parameter details per generation type.

## Example conversation

```
You: Show me the content engines for my agent

Claude: I found 2 content engines for agent "FitnessCoach":

  1. "March Content" (12 rows, 8 columns)
  2. "Viral Hooks" (5 rows, 4 columns)

You: Add a new row to "March Content" with a script about 5-minute
     ab workouts for beginners

Claude: Done. I created row 13 in "March Content" and set the Script
        cell to your text. Would you like me to generate a voiceover
        for it?

You: Yes, generate a voiceover

Claude: Generation started (id: 9001). Checking status...
        Complete! The voiceover is ready in row 13, column "Voiceover".
        URL: https://cdn.gen.pro/outputs/voice_abc123.mp3
```

## Using the API directly (without MCP)

You can also use Claude Code with the API directly via curl — just ask:

```
"Fetch my agents from the GEN API using curl.
My API key is in $GEN_API_KEY."
```

Claude will construct the appropriate curl commands, execute them, and present the results.
